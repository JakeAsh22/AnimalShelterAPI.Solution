# Animal Shelter API

#### _C#, .NET: Animal Shelter API, Jake Ash, 1/31/2020_

## Description
Create a web API for an Animal shelter of cats and dogs, and add Swagger to the API for documentation on use.

## Application should have:
- A working web api with proper calls to see, add, edit, and delete values from the API
- Swagger documentation functionality


## Setup/Installation Requirements

1. Clone this repo:
```
https://github.com/JakeAsh22/AnimalShelterAPI.Solution
```

2. Go into the repo and run this application: 
```
$ dotnet build
$ dotnet run
```

3. Check Swagger functionality:
```
- After running the application, go to http://localhost:<port>/swagger/v1/swagger.json . Then, goto http://localhost:<port>/swagger. 
- If you would like to add this API to your SwaggerHub, please make a swagger.json file, and put all of the contents from http://localhost:<port>/swagger/v1/swagger.json inside of it.  Then, choose the option with already made API's, and choose the swagger.json file as the file it should connect to.
```
